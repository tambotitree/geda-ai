Компоненты

Компонентом в схеме называется экземпляр символа из библиотеки символов. Компонент служит для представления конкретной подсхемы, либо собственно электронного компонента или устройства, которое должно использоваться в разрабатываемой схеме.

Для добавления компонента на страницу схемы нажмите I или выберите в меню Добавить→Компонент… [Add→Component…], в результате чего откроется окно «Выбор компонента» [«Select Component…»]. Окно содержит три основных области:

  • В левой части окна находится область списка библиотек — область, где можно выбрать символ, который нужно добавить на страницу.
  • В правой части окна находится область предпросмотра — здесь можно увидеть изображение выбранного символа и его атрибуты.
  • Внизу окна расположено меню режима вставки, предназначенное для выбора способа вставки компонента.
Компоненты должны добавляться только в схемы, в символы их добавлять не следует.

Выбор уже используемого компонента

Переключившись в области выбора компонентов на вкладку Используемые [In Use], можно увидеть список уже используемых в схеме символов. Для выбора символа из этого списка щёлкните по нему левой кнопкой мыши.

Если этот список нужно обновить, щёлкните по кнопке Обновить [Refresh], расположенной под списком компонентов.

Выбор компонента из библиотеки символов

Информацию о настройке того, какие библиотеки символов должны отображаться в диалоговом окне выбора компонентов, см. в разделе Настройка gschem.

Чтобы выбрать символ из библиотеки символов, щёлкните по вкладке Библиотечные [Libraries] в верхней части области выбора компонентов. При этом отобразится список имеющихся библиотек. Чтобы увидеть символы какой-либо библиотеки, дважды щёлкните по её названию или по стрелке слева от неё. Для выбора символа из списка щёлкните по нему левой кнопкой мыши.

Кроме того, можно воспользоваться поиском нужного символа среди всех доступных, набрав часть его названия в поле ввода Фильтр: [Filter], которое находится под списком библиотек.

Иногда список библиотек и символов может быть необходимо обновить (например, если вы изменили библиотеку символов, в то время как программа gschem была запущена). Для этого щёлкните по кнопке Обновить, находящейся справа под списком символов.

Добавление компонента в схему

Когда компонент выбран, для добавления его на страницу нужно щёлкнуть в области отображения схемы основного окна программы левой кнопкой мыши. Далее можно либо продолжить вставлять копии компонентов левой кнопкой, либо щёлкнуть правой для завершения вставки.

Окно «Выбор компонента» в процессе добавления компонентов можно скрыть, для этого нужно щёлкнуть по кнопке OK. Тогда при завершении добавления щелчком правой кнопки вновь будет открыто это окно.

Режимы вставки символов и внедрение

В gschem символ в схему можно вставить тремя способами:

  1. Как компонент, являющийся ссылкой на библиотечный символ (Вставить как ссылку (по умолчанию) [Default behavior - reference component]). Этот вариант установлен по умолчанию, что обеспечивает обновление всех экземпляров символа в схеме в случае обновления этого символа в библиотеке.
  2. Как компонент, данные символа для которого копируются непосредственно в схему (Внедрить в схему [Embed component in schematic]). Внедрение символа в схему облегчает его совместное использование с другими пользователями без необходимости настройки использования одинаковых библиотек символов.
  3. Как отдельные объекты, получаемые в результате разбиения символа на части (Вставить как отдельные объекты [Include component as individual objects]).

Компонент, являющийся ссылкой, можно внедрить, выделив его и выбрав в меню Правка→Внедрить компонент/изображение [Edit→Embed Component/Picture]. Преобразовать внедрённый компонент обратно в ссылочный можно с помощью Правка→Исключить компонент/изображение [Edit→Unembed Component/Picture].

Если внедрённый компонент не соответствует в полной мере какому-то символу в имеющихся библиотеках, выбор Правка→Исключить компонент/изображение не приведёт к его изменению.

Если после изменения файла исходного символа в библиотеке внедрённый компонент хочется обновить, используйте Правка→Обновить компонент [Edit→Update Component].

Редактирование символов

Чтобы изменить файл символа какого-либо компонента, выделите его и выберите пункт меню Иерархия→Войти в символ [Hierarchy→Down Symbol]. Страница этого символа будет открыта и отобразится в gschem, и её можно будет отредактировать. Для возвращения обратно в схему после завершения редактирования сохраните файл и используйте Иерархия→Наверх [Hierarchy→Up]. Может быть, для того чтобы изменения отразились в изображении схемы, вам придётся воспользоваться пунктом меню Правка→Обновить компонент.

Отсутствующие символы

Если в процессе загрузки схемы некоторые из используемых в ней символов в имеющихся библиотеках найти не получается, вместо них будут отображаться знаки предупреждения.

Просмотр документации по компоненту

При разработке символа для какого-нибудь компонента, добавьте ссылку на документацию на этот компонент с помощью атрибута documentation. См. также раздел Атрибуты.

В символе компонента может быть ссылка на спецификацию или какую-то другую относящуюся к нему документацию. Посмотреть эту документацию можно выделив компонент и выбрав в меню Справка→Документация на компонент [Help→Component Documentation].
