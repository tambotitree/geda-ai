Неявные выводы

В качестве альтернативы вычерчиванию явных выводов символов gnetlist поддерживает определение соединений между именованными соединениями и выводами с помощью атрибута «net=». Это широко используется для многосекционных компонентов во избежание необходимости добавлять к каждой секции выводы питания или чертить и добавлять в схему отдельный компонент «для соединений питания».

Подробности об использовании атрибута «net=» см. в Мини-HOWTO по атрибуту net=.
