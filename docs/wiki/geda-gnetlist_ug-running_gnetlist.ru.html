Запуск gnetlist

Вам нужна справка по установке gEDA?

У программы gnetlist простой интерфейс командной строки, и её можно запустить, набрав команду gnetlist в окне терминала. Например, обычная команда запуска gnetlist с целью получения перечня используемых в проекте компонентов может выглядеть так: 

gnetlist -g bom2 -o stepper.bom power-supply.sch motor-drive.sch

При запуске gnetlist обычно нужно указать перечень файлов схем для обработки, выбрать драйвер списка соединений, которым будет определяться выходной формат данных, и указать имя выходного файла, в котором этот список соединений будет сохранён.

gnetlist принимает множество дополнительных аргументов, которые могут использоваться для изменения его поведения. Они подробно описаны в справочной странице gnetlist(1) (которую можно посмотреть, запустив man gnetlist).
