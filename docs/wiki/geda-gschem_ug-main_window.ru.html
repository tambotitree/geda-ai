Основное окно

Эта страница содержит изображения, которые нуждаются в обновлении для соответствия последней версии gschem.

При запуске gschem открывается основное окно программы.

Окно содержит четыре основных элемента. При рассмотрении сверху вниз это будут:

  1. Панель меню, предоставляющая возможность доступа ко всем встроенным действиям по работе с файлами и редактированию.
  2. Панель инструментов, предоставляющая быстрый доступ к наиболее часто требуемым функциям.
  3. Область отображения схемы (рабочая область).
  4. Строка состояния. Слева направо в ней отображаются: текущие привязки для левой, средней и правой кнопок мыши; настройки сетки; текущая последовательность нажатий клавиш (первоначально ничего нет); и текущий режим редактирования.
